Faithful to .v4-api: three mono tabs, the active one carries an --accent underline and a tinted background. The panel reuses the memory-terminal frame, with keys in --accent, strings quiet, punctuation muted. Content is the real /.well-known/yepgent.json; the shell prompt is non-selectable so a copy stays clean. Tabs switch with a tiny inline script; with JS off, the manifest tab renders correctly.
