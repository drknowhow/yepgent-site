notes

The cursor orb is a soft --accent radial glow that trails the pointer (360px, blur 14px, opacity 0.55 dark / 0.22 light). It only appears on a fine pointer, hidden on coarse/touch input and under reduced-motion, so it is a desktop grace note, never load-bearing.

The memory field is the only heavy layer: a fixed <canvas> WebGL scene (Three.js) of drifting brand-green and neutral particles with faint constellation lines, the camera dollying through as you scroll. It runs at 0.65 opacity (0.5 light) behind a horizontal mask that drops it to ~45% across the content column, reading as a margin texture, not a backdrop. The still above is a faithful sketch of its shape, not a live simulation.

Reduced motion. Under prefers-reduced-motion: reduce, neither the cursor orb nor the WebGL field runs. The CSS/IO baseline owns the page, the aurora sits as a calm constant glow, and the scroll hairline stops scaling. CDN failure or no-WebGL degrade the same direction: quieter, never broken. Everything on this card is painted static, so it already reads true with motion off.
